💰 Expense Tracker Web Application

A full-stack Expense Tracker Web Application built using Java, Spring Boot, MySQL, Thymeleaf, Bootstrap, and Chart.js. The application helps users manage expenses, set monthly budgets, analyze spending patterns, and generate reports through an interactive dashboard.
